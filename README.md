## Option 2: Clone the Repository (RECOMMENDED ‚úÖ)

Cloning allows you to **pull the latest updates at any time** and get familiar with GitHub workflows.

### Step 1: Install Git

#### Windows

* Download Git from: [https://git-scm.com/download/win](https://git-scm.com/download/win)
* Install using default settings

#### macOS

```bash
brew install git
```

(Requires Homebrew)

Or install Git from:
[https://git-scm.com/download/mac](https://git-scm.com/download/mac)

#### Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install git
```

---

### Step 2: Clone the Repository

Open **Terminal / Command Prompt** and run:

```bash
git clone https://github.com/QuyenNV-Hanu/S2026.git
```

This will create a folder named `S2026` on your computer.

---

### Step 3: Navigate to the Repository

```bash
cd S2026
```

You can now open the folder using your IDE (IntelliJ IDEA, VS Code, etc.).
